# EvIcon

Evance's EvIcon component uses SVG-based glyphs designed within a viewBox width and height of `24px`.

    <Story of={EvIconStories.Primary} />

<ArgsTable story='Primary' />

## SVG glyphs

Evance primarily uses SVG icons.

```html
<script>
import { EvIcon } from "@evance/components";
import { Evance } from "@evance/icons";
</script>
<template>
    <ev-icon :glyph="Evance" />
</template>
```

## Webfonts
You can also use webfonts (if installed) using CSS classes.

```html
    <ev-icon class="my-icon" />
```
